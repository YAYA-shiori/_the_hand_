--------------------------
嫌ゴ「The Hand」Phase "xxxx"
--------------------------

■これはなにか
　嫌ゴです。
　FMOとSAKURA APIとNOTIFYを使って、
　同時に立っているゴーストにセクハラします。

■必要要件
　・SSP 1.10.00 (RC1)以上を強く推奨します。
　　（1.10.00で実装されたEXECUTE SSTP GetCollision拡張を利用しています）
　・上記以外の場合（Materia等） Sakura API を利用しますが
　　あまり動作確認していません。
　・同時に別のゴーストが立っていないと意味がありません。
　・結構重い処理をするゴーストですのでお気をつけください。

■機能
　ダブルクリック　触りに行く
　Ｓキー　　　　触りをやめる
　Ｒキー　　　　　ランダムトーク
　Ｃキー　　　　　コミュニケート
　Ｍキー　　　　　メニュー
　Ｘキー　　　　　必殺技１

■言い訳
　・このゴーストは「当たり判定」域利用の技術的デモを目的として作成されました。
　・当然ですがこのゴーストと作者の品性はまったく関係しません。
　　わたしこんな下品じゃないです。
　・ていうか存在自体が冗談みたいなゴーストですごめんなさい許してください。
